# Button

A button component in a user interface is a graphical element that users can interact with by clicking or tapping. Buttons are a fundamental part of UI design and serve various purposes, such as triggering actions, submitting forms, or navigating between different sections of an application.

<br />
<br />

#### Installation

<CH.Code>
```bash npm
npm install @codecentrum/piksel-ui
```

```bash yarn
yarn add @codecentrum/piksel-ui
```

```bash pnpm
pnpm add @codecentrum/piksel-ui
```

```bash bun
bun add @codecentrum/piksel-ui
```
</CH.Code>

```tsx page.tsx
import { Button } from '@codecentrum/piksel-ui'
```
