# Button

A button component in a user interface is a graphical element that users can interact with by clicking or tapping. Buttons are a fundamental part of UI design and serve various purposes, such as triggering actions, submitting forms, or navigating between different sections of an application.

<br />

#### Installation

<CH.Code>
```bash npm
npm install @codecentrum/piksel-ui
```

```bash yarn
yarn add @codecentrum/piksel-ui
```

```bash pnpm
pnpm add @codecentrum/piksel-ui
```

</CH.Code>

```tsx page.tsx
import { Button } from '@codecentrum/piksel-ui'
```

<br />

#### Example

<div className="flex items-center space-x-4 mt-4">
  <Button>Continue</Button>
  <Button theme="secondary">Cancel</Button>
  <Button theme="destructive">Delete</Button>
</div>

##### Variant
Use the variant prop to control the visual style of the button.

<br />


###### Soft

<div className="flex items-center space-x-4 mt-4">
  <Button variant="soft">Continue</Button>
  <Button theme="secondary" variant="soft">Cancel</Button>
  <Button theme="destructive" variant="soft">Delete</Button>
</div>

<br />


```tsx soft.tsx
<Button variant="soft">Continue</Button>
<Button theme="secondary" variant="soft">Cancel</Button>
<Button theme="destructive" variant="soft">Delete</Button>
```

##### Size
Use the size prop to control the size of the button.


<div className="flex items-center space-x-4 mt-4">
  <Button size="small" theme="secondary">Small</Button>
  <Button theme="secondary">Medium</Button>
  <Button size="large" theme="secondary">Large</Button>
</div>
