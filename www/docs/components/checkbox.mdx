# Checkbox

A control that allows the user to toggle between checked and not checked.

<br />

#### Installation

<CH.Code>
```bash npm
npm install @codecentrum/piksel-ui
```

```bash yarn
yarn add @codecentrum/piksel-ui
```

```bash pnpm
pnpm add @codecentrum/piksel-ui
```

</CH.Code>

```tsx page.tsx
import { Checkbox } from '@codecentrum/piksel-ui'
```

<br />

#### Example

<div className="mt-4">
  <Checkbox id="agreement" label="I agree with terms and condition" />
</div>

```tsx example.tsx
<Checkbox id="agreement" label="I agree with terms and condition" />
```
