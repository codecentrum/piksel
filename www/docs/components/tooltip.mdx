# Tooltip

A popup that displays information related to an element when the element receives keyboard focus or the mouse hovers over it.

<br />

#### Installation

<CH.Code>
```bash npm
npm install @codecentrum/piksel-ui
```

```bash yarn
yarn add @codecentrum/piksel-ui
```

```bash pnpm
pnpm add @codecentrum/piksel-ui
```

</CH.Code>

```tsx page.tsx
import { Tooltip } from '@codecentrum/piksel-ui'
```

<br />

#### Example

<div className="mt-4">
  <Tooltip
    trigger={<Badge variant="soft">Hover Me</Badge>}
    content="This is an explanation"
  />
</div>


```tsx page.tsx
<Tooltip
  trigger={<Badge variant="soft">Hover Me</Badge>}
  content="This is an explanation"
/>
```
