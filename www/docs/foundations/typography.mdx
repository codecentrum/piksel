# Typography

visual representation and styling of text within the application or website. Typography is a crucial aspect of design that not only involves choosing fonts but also encompasses factors such as font size, line spacing, letter spacing, and text color.

<br />

#### Usage
Headings in HTML h1-h6 represent hierarchical levels of titles or headings on a web page. By default, these heading tags have predefined styles, but they can be further styled using CSS to match the design so we only using the tag HTML
